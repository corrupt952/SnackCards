# Snack Cards

Beautiful card view for your Chrome Reading List in a side panel. Articles and videos, simple and clean.

[Chrome Web Store](https://chromewebstore.google.com/detail/snack-cards-reading-list/pegiigaifjibokehgiofjhndlkfnocoe)

## Development

### Prerequisites

- [mise](https://mise.jdx.dev/) or compatible mise.toml file

### Setup

1. Install Node.js

    ```bash
    mise install
    ```

1. Install pnpm

    ```bash
    npm install -g pnpm
    ```

1. Install dependencies

    ```bash
    pnpm install
    ```

1. Run the development server

    ```bash
    pnpm dev
    ```

1. Load the [extension](chrome://extensions/) from the `.output/chrome-mv3-dev` directory
