# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [Unreleased]

## [2026.02.1] - 2026-02-10

### Added
- Search by title in the side panel

### Changed
- Simplified to a plain list model (removed unread/read filter tabs and toggle buttons)
- Badge now shows total item count in neutral color instead of unread count in red
- Updated extension description for Chrome Web Store

### Removed
- Unread/read filter tabs (Unread / All / Read)
- Mark as read / Mark as unread toggle buttons
- Unread count badge in header

## [2025.10.1] - 2025-10-01

### Added
- Initial release of Snack Cards
- Display Chrome Reading List in a side panel
- Filter reading list items by all/unread/read status
- Mark articles as read/unread
- Remove articles from reading list
- Auto-mark articles as read when opened
- Open articles in current tab (matching Chrome's native behavior)
- Unified card layout for articles and videos
- Video thumbnail display with YouTube/Vimeo/TikTok support
- Dark mode support
